## Purpose

Demonstrate display of a Google Map in a React Component with additional marker(s).

## Running

In the project directory, you can run:

  `npm start`

Then navigate to `localhost:3000`.

## Other Commands

  * `npm test`
  * `npm run build`
  * `npm run eject`
